# Comment Doc Lens Language Service Fixtures

These fixtures are minimal workspaces for collecting real language-service evidence for experimental adapters.

Open each subdirectory as the VS Code workspace root, install the recommended extensions from `.vscode/extensions.json`, then run:

1. `Comment Doc Lens: Show Language Status`
2. `Comment Doc Lens: Copy Diagnostics for Issue`
3. `Comment Doc Lens: Refresh`

Record the output in `docs/language-service-evidence.md`. Keep these fixtures small and dependency-light so evidence can be reproduced on a clean machine.
